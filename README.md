# Tratamento-de-Dados-de-forma-Pratica-utilizando-a-linguagem-Python
Este repositório contém um projeto de análise exploratória e tratamento de dados aplicado a um dataset climático. Foi desenvolvido em Python, utilizando as bibliotecas:  Pandas  NumPy  Matplotlib  O objetivo é transformar uma base bruta em um conjunto de dados limpo, consistente e pronto para análises posteriores.
